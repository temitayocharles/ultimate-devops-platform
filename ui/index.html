Ultimate DevOps Platform

Portable DevOps workstation container with batteries included.

Quick Start

docker run --rm -it \
  -v /var/run/docker.sock:/var/run/docker.sock \
  -v $HOME/devops-work:/home/devops/work \
  -e UI_ENABLED=true \
  -p 8080:8080 \
  temitayocharles/ultimate-devops-platform:latest

Core Tools

  • kubectl, helm, k9s, kustomize
  • terraform, terragrunt, packer
  • vault, argocd, flux
  • aws cli, jq, yq
  • go, node, python

Optional UI

Enable with UI_ENABLED=true. Serves this page on port 8080.

Logs: /tmp/ui.log

Docker Engine

Use host socket (recommended) or enable Docker-in-Docker with ENABLE_DOCKERD=true.
